: Code Obfuscation

  Code obfuscation is a technique for making source code more complex and therefore harder to understand. Through the SDK, VxLang modifies the code in user-specified regions to make it harder to understand.

: How to use obfuscation

  Code Obfuscation can be positioned and scoped as desired by the user using the VxLang SDK.
  The default code obfuscation is bounded by VL_OBFUSCATION_BEGIN and VL_OBFUSCATION_END. This method adds dummy code in between your code, making it harder to track your code.
